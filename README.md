# kubernetes build
`enable kubernetes in docker desktop`
`make sure both docker and kubernetes is running`
`type kubectl in terminal`
`create pod.yaml`
`kubectl get pods`
`kubectl get nodes`
`kubectl apply -f pod.yaml`
`create service-local.yaml`
`kubectl apply -f service-local.yaml`
`kubectl get service`
